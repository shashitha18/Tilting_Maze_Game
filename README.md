## Introduction 🚀

The Tilting Maze Game is a fun and challenging game where the objective is to move all the balls to the center of the maze. You control the movement of the balls by tilting the maze using a joystick. The game offers different modes and customization options to enhance your gaming experience.

## Features ✨

- Intuitive joystick control for tilting the maze 🕹️
- Realistic physics simulation for ball movement 🎱
- Multiple game modes: Easy and Hard 🎯
- Customizable colors for maze elements 🎨
- Interactive gameplay with smooth animations 🔄

## How to Play ℹ️

1. Click on the joystick to start the game.
2. Tilt the maze using the joystick to move the balls.
3. Move all the balls to the center of the maze to win.
4. Avoid black holes in Hard mode to prevent losing the game.
5. Press 'H' for Hard mode and 'E' for Easy mode anytime during the game.

## Controls 🎮

- Use the joystick to tilt the maze.
- Press 'H' for Hard mode.
- Press 'E' for Easy mode.
- Press Spacebar to restart the game.

## Game Modes 🎯

### Easy Mode

In Easy mode, your objective is to move all the balls to the center of the maze. There are no obstacles to hinder your progress.

### Hard Mode

In Hard mode, you must avoid black holes while moving the balls to the center of the maze. If any ball falls into a black hole, you lose the game.

## Customization 🎨

You can customize the colors of various elements in the maze by modifying the CSS variables in the `style.css` file. Simply adjust the values of the variables to change the colors.

Simply open the `index.html` file in your web browser to start playing the Tilting Maze Game. Use the joystick to tilt the maze and move the balls accordingly.
